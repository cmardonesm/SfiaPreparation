Comandos para subir cambios: * Verificar estado de archivos, rojos sin agregar y verdes agregados a commit git status * Agrega todos los cambios a commit git add . * creacion de commit git commit -m "mensaje descriptivo indicando cambio" * consolida cambios en repositorio git push Comandos para halar cambios del repositorio al area local git pull Si falla el pull, seguir las intrucciones que despliega la linea de comandos (Basicamente pedira hacer un commit y un push de los cambios antes de poder consolidar el pull)
